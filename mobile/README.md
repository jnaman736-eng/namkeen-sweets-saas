# Namkeen & Sweets Mobile App

Full-featured React Native mobile application for managing namkeen and sweets manufacturing across multiple locations.

## Features

- 📱 Cross-platform (iOS & Android)
- 🔐 Secure authentication
- 📦 Real-time order management
- 📊 Inventory tracking
- 📍 Multi-location support
- 🔔 Real-time notifications
- 📈 Analytics dashboard
- 🔄 Real-time synchronization

## Installation

```bash
cd mobile
npm install
```

## Running the App

### Development
```bash
npm start
```

### Android
```bash
npm run android
```

### iOS
```bash
npm run ios
```

## Project Structure

```
mobile/
├── screens/        # Screen components
├── components/     # Reusable components
├── navigation/     # Navigation configuration
├── store/          # Redux store
├── api/            # API endpoints
├── utils/          # Utility functions
├── styles/         # Theme and styles
└── App.js          # Entry point
```

## Building for Production

```bash
npm run build
```
